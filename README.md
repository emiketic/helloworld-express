## Requirements

- MongoDB v3.6
- Redis v4

## Usage

```sh
# install dependencies
npm install

# lint code for critical issues
npm run lint:critical

# lint code
npm run lint

# format code
npm run format

# run tests
npm run test

# run api worker
npm run app:api

# run job runner
npm run app:job

# run console
npm run app:console
```
